# APP NEXTjs

## Pré-requisitos

Antes de iniciar, certifique-se de possuir instalado em seu ambiente:

* Node.js 
* Yarn ou NPM

## Instalação e Configuração

Siga os passos abaixo para configurar o ambiente local:

### 1. Instalar dependências do NEXTJS

```bash
npm install
```

ou 

```bash
yarn install
```

Crie e edite o arquivo `.env` e configure:

* URL da API 

Para subir o projeto em uma porta diferente da padrão acesse o package.json e altere o script: dev para = "next dev -p <porta_desejada>"
